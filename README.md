# 千牛话术助手

一个轻量的千牛接待台悬浮话术工具。

## 当前功能

- 无系统边框悬浮窗；
- 自动吸附在千牛接待台右侧；
- 可手动拖动窗口；
- 点击最小化后隐藏为可拖动圆形图标；
- 点击圆形图标恢复主界面并重新吸附千牛；
- 点击话术后自动填充到千牛客服输入框；
- 支持分组；
- 支持自定义分组名称；
- 支持新增、编辑、删除分组和话术；
- 话术用 `phrases.txt` 文本文件保存，放在工具根目录；
- 如果只复制 `QianniuAssistant.exe` 到新电脑/新目录运行，程序会自动在 exe 同目录创建预置 `phrases.txt`。

## 使用方式

1. 打开千牛接待台。
2. 双击运行 `QianniuAssistant.exe`。
   - 如果同目录没有 `phrases.txt`，程序会自动创建预置话术文件。
3. 先把光标点进千牛客服输入框，保持输入框处于编辑状态。
4. 点击话术卡片后，工具窗口不会主动抢焦点，内容会直接粘贴到原本正在编辑的千牛输入框。
5. 如果没有找到千牛聊天窗口，工具会保留复制到剪贴板的内容，可手动粘贴。
6. 点击「编辑」可维护分组和话术。

## phrases.txt 格式

```txt
[问候语]
欢迎光临 = 亲，欢迎光临！有什么可以帮您的吗？
感谢咨询 = 感谢咨询，祝您生活愉快！

[物流相关]
发货时间 = 亲，我们会尽快为您安排发货哦~
```

说明：

- `[分组名]` 表示一个分组；
- `标题 = 话术内容` 表示一条话术；
- 也可以只写一行话术，不带等号，系统会自动用前20个字作为标题；
- 保存后点击工具内「刷新」即可重新读取。

## 快捷操作

- 双击标题栏：切换自动吸附 / 手动模式；
- 拖动标题栏：进入手动模式；
- 点击最小化按钮：隐藏为圆形图标；
- 拖动圆形图标：移动隐藏图标位置；
- 点击圆形图标：展开工具并恢复吸附；
- `F5`：刷新 `phrases.txt`；
- `Esc`：关闭工具。

## 打包

```bash
pyinstaller build.spec
```
